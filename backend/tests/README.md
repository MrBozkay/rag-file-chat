# Test Infrastructure

Test dosyaları oluşturuldu ancak henüz çalıştırılmadı.

## Test Dosyaları
- `conftest.py` - Pytest fixtures
- `test_routes.py` - API endpoint testleri
- `test_models.py` - Database model testleri

## Testleri Çalıştırma

```bash
cd backend
source venv/bin/activate
pytest tests/ -v
```
